`Jetti` can find `.proto` files and run `protoc` command to build protobuf files.

Also, `Jetti` can find `.fbs` files and run `flatc` command to build flatbuffer files.

## Install compiler

You can install protobuf compiler in [here]("https://grpc.io/docs/languages/go/quickstart/").

And, you can install flatbuffers compiler in [here]("https://google.github.io/flatbuffers/flatbuffers_guide_building.html").

Or, you can install them with `brew`, `pacman`, `apt`, `dnf`, etc...

## Build protobuf

Run `jetti new model/person/person`, then `./model/person/person.proto` file will be generated.

```proto
syntax = "proto3";

package person;

option go_package = "model/person/person";
```

And append some fields to `./model/person/person.proto` file.

```proto
syntax = "proto3";

package person;

option go_package = "model/person/person";

message Person {
  string name = 1;
  int32 age = 2;
}
```
